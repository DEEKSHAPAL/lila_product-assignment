LILA BLACK Player Journey Visualizer

A Streamlit-based telemetry tool for exploring player and bot routes, combat hotspots, death locations, storm deaths, loot pickups, and match flow across production gameplay data.

GitHub Repository Open Interactive Dashboard
This Vercel URL is a project landing page. It is not the interactive Streamlit dashboard.

How to launch the real app

The live dashboard is deployed on Streamlit Community Cloud. Use that URL for the interactive map, timeline, filters, heatmaps, and CSV download.

  1. Open the interactive dashboard from the button on this page.
  2. Use the GitHub repository button to review the full source code.
  3. Run locally with streamlit run app.py if a local review is needed.

Local run command:
streamlit run app.py
